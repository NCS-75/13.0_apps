Google contacts synchronization

This application facilitates users and managers to synchronize their Google account related contacts with Odoo.

Users can import, export, update and delete contacts between Google contacts and Odoo.

Synchronization facilitates users to keep their Google contacts updated in their Odoo account.

This module is also compatible with our odoo v13 Allure Theme

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Note: please check file 'README.rst' for required python package installation.

Key Features

    Import Google contacts in Odoo
    Export Odoo contacts to Google
    Update contacts while import from Google to Odoo
    Update contacts while export from Odoo to Google
    Delete Contacts functionality

Setup in Google

To use Google contacts API users need to generate Google Client ID and Google Client Secret.

Steps are as following

Go to https://console.developers.google.com

Sign In with your Google account.

Create New Project


Enable Contacts API


Create Credentials and select 'Oauth client ID'.


Configure Consent Screen.


Create a Client ID. Enter the name of the application and the allowed pages on which you will be redirected.

Setup in Odoo

Install Google Contact Synchronization App from the Apps menu

Go to Settings -> General Settings and enter your Client ID and Client Secret in Google Contacts option.

Export Contacts

Go to Google Contact -> Sync Contacts and click 'Export Contact' button.

Odoo Contacts


Odoo contacts are exported in related user's Google account successfully.

Import Contacts

Contacts in Google


Go to Google Contact -> Sync Contacts and click 'Import Contact' button.


Imported contact in Odoo

Scheduler Action

From Odoo to Google 'Export Google Contacts' scheduler will run at configured interval time and will export all contacts to Google from Odoo automatically.


From Google to Odoo 'Import Google Contacts' scheduler will run at configured interval time and will import all contacts from Google to Odoo automatically.
